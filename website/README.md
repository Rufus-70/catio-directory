# Catio Directory Website

**Stack**: Next.js + Supabase + Cloudflare Workers
**Deploy time**: 20 minutes
**Cost**: $0 (free tiers)

## Quick Deploy

```bash
cd ~/catio-directory/website
npm install
npm run build
npx wrangler deploy
```

## Features

- ✅ Search by city/state
- ✅ Filter by catio type (window box, patio, standalone, etc.)
- ✅ Business listing pages
- ✅ Portland Catios featured at top
- ✅ Mobile responsive
- ✅ SEO optimized
- ✅ Fast (Cloudflare edge network)
